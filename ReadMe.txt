compile program with: make
run program: ./PMan

commands supported
-bg (example program)					//runs specified program in the background
-bglist							//lists all background programs started with PMan
-bgkill (pid)						//kills the specified process
-bgstop (pid) 						//stops the specified process
-bgstart (pid)						//starts the specified process
-pstat (pid)						//lists info about the specified process
-exit

PMan is a program that allows you to run programs in the background while still providing
you a prompt to execute any of the different commands supported.
